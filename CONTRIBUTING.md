# Contributing

`kaiokai` is the public theme-only package for this project.

## Principles

- Prefer correctness over approximation.
- Keep this repo focused on the Zed theme package itself.
- Avoid adding local tooling, extracted assets, or unrelated experiments to the public repo.
- Keep editor-behavior fixes separate from the theme when they belong upstream in Zed or in a dedicated language extension.

## Scope

Changes are usually a good fit here when they improve one of these:

- theme colors and syntax presentation
- syntax mapping coverage for Zed's capture model
- packaging, installation, and repo maintenance

Generation tooling and icon workflows live outside this repo.
